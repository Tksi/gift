# Product Overview

**No Thanks!** (ノーサンキュー) ボードゲームのオンライン対戦プラットフォーム。

## Core Capabilities

- **リアルタイムゲームセッション管理**: 複数プレイヤーによる同時対戦をサポート
- **Server-Sent Events (SSE) によるリアルタイム配信**: 状態変更の即時通知
- **ターン制ゲーム進行**: チップ配置・カード取得のアクション管理
- **イベントログ・監視機能**: ゲーム履歴の追跡とエクスポート

## Target Use Cases

- 友人同士でのオンライン対戦（2〜7名）
- ゲームルールに基づいたヒント提供
- ゲーム履歴の確認・分析

## Value Proposition

- シンプルなカードゲームを Web ベースでプレイ可能に
- リアルタイム同期による快適なマルチプレイ体験
- TypeScript による堅牢な型安全性とテスト駆動開発
